Práctica

Diseña una clase en Python llamada Polinomio para representar polinomios de grado $2$ con coeficientes enteros, esto es polinomios de la forma $$a_{0}+a_{1}x + $a_{2}x^2,$$ donde $a_{0}, a_{1}, a_{2} \in \mathbb{Z}$. Esta clase debe implementar métodos para realizar operaciones algebraicas básicas (suma, resta, multiplicación) y evaluar el polinomio para un valor dado de la variable. La clase debe cumplir con las siguientes caracteristicas:

  1. Constructor de la clase (__init__)
    • Debe aceptar como coeficientes sólo valores enteros, en el siguiente orden: $a_{0}$, $a_{1}, $a_{2}$.
    • Si no se proporciona un valor para $a_{2}$, asumir que el valor correspondiente es $1$
    • Si el valor ingresado para $a_2 = 0$, lanzar un ValueError con el un mensaje apropiado, indicando que el valor del ccoeficiente $a_2$ no puede ser cero porque se trata de un polinomio de grado $2$.
  • Métodos de instancia obligatorios
    • evaluar(self, x): Devuelve el valor del polinomio evaluado en $x$
    • derivada(self): Devuelve un nuevo objeto Polinomio que representa la derivada del polinomio actual.
    • suma(self, otro_polinomio): Devuelve un nuevo polinomio que es la suma del polinomio actual con otro_polinomio.
